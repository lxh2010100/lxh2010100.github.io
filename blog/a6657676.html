1 更换国内源

Ubuntu访问一些国外的网站，软件源等等，速度都会比国内的网站要慢。我们可以更新软件源，或镜像源达到加速的目的

  1. 适用系统 Ubuntu22.4,如果是别的版本的Ubuntu，源可能不同，自行百度更换，方法相同
jack@jack-HP:~$ uname -a
Linux jack-HP 5.19.0-35-generic #36~22.04.1-Ubuntu SMP PREEMPT_DYNAMIC Fri Feb 17 15:17:25 UTC 2 x86_64 x86_64 x86_64 GNU/Linux
  1. 备份原先的官方源
jack@jack-HP:~$ sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup
  1. Sources.list 填充以下内容,阿里云镜像源

2.1 安装vim

sudo apt install vim -y

"设置按照语法方式折叠（可简写set fdm=XX）"
"有6种折叠方法：
"manual   手工定义折叠"
"indent   更多的缩进表示更高级别的折叠"
"expr     用表达式来定义折叠"

"zc C-lose a fold (关闭折叠)
"zf F-old creation (创建折叠)
"nnoremap <space> @=((foldclosed(line('.')) < 0) ? 'zc' : 'zo')<CR>

""""""""""""""""""""""""""""""""
"""=>在shell脚本开头自动增加解释器以及作者等版权信息<="""
""""""""""""""""""""""""""""""""
"新建.py,.cc,.sh,.java文件，自动插入文件头"
autocmd BufNewFile *.py,*.cc,*.sh,*.java exec ":call SetTitle()"
"定义函数SetTitle，自动插入文件头"
func SetTitle()
    if expand ("%:e") == 'sh'
        call setline(1, "#!/bin/bash")
        call setline(2, "#Author:bert")
        call setline(3, "#Blog:http://blog.51cto.com/zpf666")
        call setline(4, "#Time:".strftime("%F %T"))
        call setline(5, "#Name:".expand("%"))
        call setline(6, "#Version:V1.0")
        call setline(7, "#Description:This is a production script.")
    endif
endfunc

3 版本管理

3.1 安装git

sudo apt install git -y

3.2 配置git

1 配置用户名密码

git config --global user.name "xxx"
git config --global user.email "xxx.com"

通过以下命令查看配置清单

git config --list

2 创建ssh秘钥

ssh-keygen -t rsa -C "xxx.com"

一路回车，直至结束

3 公钥上传至个人github gitee账号

cat ~/.ssh/id_rsa.pub

github: 头像 –> Settings –> SSH and GPG keys –> New SSH key

gitee: 头像 –> 账号设置 –> ssh公钥 –> 添加公钥

这样clone 或者github/gitee网站的代码时就不用输入用户名密码了。

4 编译

4.1 安装gcc/gdb

sudo apt install gcc gdb -y

4.2 安装交叉编译器

sudo apt install g++-aarch64-linux-gnu -y
aarch64-linux-gnu-gcc -v  #查看默认安装的gcc版本
aarch64-linux-gnu-g++ -v  #查看默认安装的g++版本

4.3 编译工具

sudo apt install make cmake cmake-gui -y #安装 cmake-gui 有些需要选择性编译的库使用较方便

5 远程开发

远程机：Ubuntu22.04

本地：windows/macOS

5.1 安装ssh

  1. 安装net-tools，查看Ubuntu的ip地址
sudo apt install net-tools -y
ifconfig
  1. 安装ssh
sudo apt install openssh-server -y
sudo /etc/init.d/ssh start
ps -e | grep ssh  # 7497 ?        00:00:00 sshd
  1. 本地连接
ssh -p [port] [user name]@[remote ip]

⭐⭐⭐ port端口一般是默认是22，可不用-p指定。如果是其它端口，需要使用-p指定

5.2 配置本地免密登陆

1 获取==本地windows/macOS公钥==id_rsa.pub，并复制其中的内容

2 新建文件authorized_keys,并将刚才复制的公钥粘贴到Ubuntu。wq保存退出

vim ~/.ssh/authorized_keys

这样下次在本地ssh连接Ubuntu时就不需要每次都输入密码了

6 python

6.1 安装python2.7

sudo apt install python2.7 -y
jack@jack-HP:~$ python2.7 -V
Python 2.7.18

6.2 安装python3

推荐使用python3

sudo apt install python3 -y # 默认python3版本
sudo apt install python3.11 -y # 指定python3版本
jack@jack-HP:~$ python3 -V
Python 3.10.6 # 默认python3版本

6.3 设置python3为默认

jack@jack-HP:~$ sudo ln -s /usr/bin/python3 /usr/bin/python
jack@jack-HP:~$ which python
/usr/bin/python
jack@jack-HP:~$ python -V
Python 3.10.6

这样在终端中输入python，python解释器默认就是python3了。😄

6.4 安装pip3

sudo apt install build-essential libssl-dev libevent-dev libjpeg-dev libxml2-dev libxslt-dev # 安装build依赖包 
sudo apt install python3-pip -y
sudo apt install python3-tk -y # use for import
jack@jack-HP:~$ pip -V
pip 22.0.2 from /usr/lib/python3/dist-packages/pip (python 3.10)

6.5 Pip3安装python库

  1. 一般安装，这种方式访问国外源，安装会非常慢，还有可能安装失败
pip install Pandas
  1. 安装时手动指定镜像源
pip install Pandas -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com
  1. 配置默认源
mkdir -p ~/.pip
sudo vim ~/.pip/pip.conf
# add
[global]
index-url = https://mirrors.aliyun.com/pypi/simple/

[install]
trusted-host=mirrors.aliyun.com

清华：https://pypi.tuna.tsinghua.edu.cn/simple/
中国科技大学 ：https://pypi.mirrors.ustc.edu.cn/simple/
华中科技大学：http://pypi.hustunique.com/
豆瓣：http://pypi.douban.com/simple/

8 expect工具

expect是一个自动化交互套件，主要应用于执行命令和程序时，系统以交互形式要求输入指定字符串，实现交互通信。

sudo apt install expect -y

9 Linux内核qemu

  1. 命令行安装
sudo apt-get install qemu-system -y # 安装
sudo apt install -y qemu-system-arm # 只安装arm
  1. 手动安装
sudo apt-get install ninja-build
sudo apt-get install pkg-config

tar xvJf qemu-7.2.0.tar.xz
cd qemu-7.2.0
./configure
make -j 64
